Tutorial Steps

Step 1. 레포지토리 생성

Step 1. Create Repository

GitHub 계정에서 새로운 레포지토리를 생성하세요. 이름은 자유롭게 정하되 'Public'으로 설정해야 GitHub Pages 이용이 쉽습니다.

Create a new repository on GitHub. Name it as you like, but ensure it's 'Public' for easy GitHub Pages setup.

Step 2. 코드 작성 (Prompt)

Step 2. Writing Code (Prompt)

AI를 활용해 미니멀한 코드를 생성할 때 아래의 대표 프롬프트를 사용해 보세요.

Use the representative prompt below when generating minimal code with AI.

"Create a minimal one-page HTML with CSS for a portfolio, including light/dark mode and responsive design."

Step 3. 배포하기

Step 3. Deployment

파일을 업로드한 후, Settings > Pages 탭에서 Build and deployment 설정을 'Deploy from a branch'로 선택하고 저장하세요.

After uploading files, go to Settings > Pages and select 'Deploy from a branch' under Build and deployment.
